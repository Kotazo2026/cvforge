# CVForge

Générateur de CV premium avec éditeur live, 5 templates et export PDF.

## Prérequis

- Node.js 20+
- npm

## Installation

```bash
npm install
```

## Développement

```bash
npm run dev
```

Ouvrir [http://localhost:3000/editor](http://localhost:3000/editor).

## Scripts

| Commande        | Description              |
| --------------- | ------------------------ |
| `npm run dev`   | Serveur de développement |
| `npm run build` | Build production         |
| `npm run lint`  | ESLint                   |
| `npm run test`  | Tests Vitest (Bloc 2+)   |
